Protect the QSO database

Option 1 — Bookmark this page (easiest)

Press Ctrl+D (Windows/Linux) or ⌘D (Mac) to bookmark this page. Firefox automatically protects IndexedDB storage for bookmarked origins — no settings change needed.

Option 2 — Firefox storage exception

Go to Settings → Privacy & Security → Cookies and Site Data → Manage Exceptions and add as Allow.

Option 3 — Regular backup

Use the Backup / Restore section below to export a JSON file after each session.
