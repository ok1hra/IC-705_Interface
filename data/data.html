JS8CALL OVER WLAN

IC-705 setup

The radio is not in a suitable data mode. Complete the setup below and load the JS8CALL WLAN preset; the JS8Call header must show USB-D.

  1. Prepare the radio and station.

    Use IC-705 firmware 1.20 or newer, which includes the PRESET function. Connect a suitable antenna or dummy load, select a low RF power for initial tests, and synchronize the phone or computer system clock.

  2. Connect the IC-705 to WLAN.

    Open MENU → SET → WLAN Set, switch WLAN on, and connect the radio to the same network as this interface. Direct IC-705 access-point mode can also be used.

  3. Enable network control.

    In MENU → SET → WLAN Set → Remote Settings, set Network Control to ON, keep the default control/serial/audio ports 50001 / 50002 / 50003, and configure a Network User ID and password with administrator access. Enter the same radio address and credentials on this device's SETUP page, then restart the IC-705 if requested.

  4. Route receive audio to WLAN.

    Set MENU → SET → Connectors → WLAN AF/IF Output → Output Select to AF and AF SQL to OFF (Open).

  5. Create a dedicated preset.

    Open page 2 of MENU → PRESET. Long-press an unused memory, choose Edit the Preset Memory, and name it JS8CALL WLAN. Select the check boxes for the settings listed in the next step.

  6. Enter the JS8Call preset values.

    Finish with <<Write>> → YES. Separately set Connectors → MOD Input → WLAN MOD Level to 50% as a safe starting point. DATA OFF MOD may remain on the normal microphone input so voice operation still works outside data mode.

  7. Load and verify the preset.

    Open MENU → PRESET → JS8CALL WLAN → YES. The preset must show In Use, and this page's header must report USB-D. If it reports CW, RTTY, AM, FM, or another mode, reload the preset.

  8. Complete the JS8Call page setup.

    Choose a JS8 dial frequency by tapping the TRX frequency, set My callsign and My grid, wait for the audio clock to lock, and confirm Enable radio TX. Use TUNE briefly and adjust TX audio gain or WLAN MOD Level for a clean signal with little or no ALC indication.

Reminder: JS8Call is a high-duty-cycle digital mode. Use conservative RF power and verify antenna matching before transmitting.

Source: Icom IC-705 manuals and JS8Call user guide
!

JS8Call requires TRX1 over LAN

This function uses Icom network control and bidirectional WLAN audio. It is not available with a Bluetooth or serial/CAT connection.

  1. On the radio, connect to WLAN and enable Network Control in its remote/network settings.
  2. Open SETUP → Radio and select TRX1 Connection: LAN.
  3. Enter the radio IP address, network username and network password, then save the setup.

IC-705 is tested. Other Icom transceivers using the compatible LAN remote-control and audio protocol may also work.
